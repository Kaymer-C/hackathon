Setup guide

Running on your laptop in 5 steps

01

Clone the repo & open in VS Code

# In your terminal
git clone https://github.com/yourteam/peerprep
cd peerprep
code .

VS Code will prompt you to install recommended extensions — click Install All.

02

Install dependencies

npm install
pip install RPi.GPIO paho-mqtt requests --break-system-packages

The root package.json installs both the client and server with one command.

03

Copy .env and add secrets

cp .env.example .env
# Edit .env:
JWT_SECRET=your_secret_here
MQTT_BROKER=localhost
S3_BUCKET=your-bucket-name

The JWT secret must match across all three modules. Share it at the 11 AM sync.

04

Start all services

npm run dev
# Starts: React client (3000), Node server (3001), coturn

# On the Pi (separate terminal):
python firmware/pod_controller.py

Open http://localhost:3000 in your browser to see the dashboard.

05

Run the smoke test

npm test
npm run audit
# Should show: 0 critical vulnerabilities

If all tests pass, you're ready to record a peer interview session. Press the pod button to start.
